Once again, here we are with another NiFi release voting process.  NiFi 1.23.1 is ready.  At this point after 3 releases, the process is easy for me to duplicate.   Below are the steps I executed on my machine to build and test NiFi 1.23.1:


```ruby
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.1/nifi-1.23.1-source-release.zip
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.1/nifi-1.23.1-source-release.zip.asc
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.1/nifi-1.23.1-source-release.zip.sha256
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.1/nifi-1.23.1-source-release.zip.sha512
gpg --verify -v nifi-1.23.1-source-release.zip.asc
shasum -a 256 nifi-1.23.1-source-release.zip
shasum -a 512 nifi-1.23.1-source-release.zip
unzip -DD nifi-1.23.1-source-release.zip
cd nifi-1.23.1
JAVA_HOME=`/usr/libexec/java_home -v 17` mvn clean install -Pcontrib-check
cd nifi-assembly/target/
unzip nifi-1.23.1-bin.zip
./nifi-1.23.1/bin/nifi.sh start
tail -f ./nifi-1.23.1/logs/nifi-app.log
```
