I have been so busy working on customer projects and with my intern that I have not had time to blog.   As time is flying by, here we are with another NiFi release voting process.  NiFi 1.23 is ready.  Here are the steps I executed on my machine to build and test NiFi 1.23


```ruby
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.0/nifi-1.23.0-source-release.zip
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.0/nifi-1.23.0-source-release.zip.asc
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.0/nifi-1.23.0-source-release.zip.sha256
wget https://dist.apache.org/repos/dist/dev/nifi/nifi-1.23.0/nifi-1.23.0-source-release.zip.sha512
gpg --verify -v nifi-1.23.0-source-release.zip.asc
shasum -a 256 nifi-1.23.0-source-release.zip
shasum -a 512 nifi-1.23.0-source-release.zip
unzip -DD nifi-1.23.0-source-release.zip
cd nifi-1.23.0
JAVA_HOME=`/usr/libexec/java_home -v 17` mvn clean install -Pcontrib-check
cd nifi-assembly/target/
unzip nifi-1.23.0-bin.zip
./nifi-1.23.0/bin/nifi.sh start
tail -f ./nifi-1.23.0/logs/nifi-app.log
```
